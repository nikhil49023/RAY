# ⚡ RAY God Mode

Your scientific research assistant with manual brain control.

### Quick Start
1. Click the **⚙️ Settings** icon (top right)
2. Select your **Model** (Groq, Sarvam, Ollama, or OpenRouter)
3. Choose a **Mode** (Standard, Research, or Reasoning)
4. Start asking questions

### Features
- 🧠 **Multiple Models** — Groq, Sarvam M1, Ollama, OpenRouter (free models)
- 🔍 **Web Search** — DuckDuckGo for quick facts
- 🕷️ **Deep Research** — Self-hosted Firecrawl for full domain crawling
- 📄 **Canvas** — Long documents open in a side pane
- 📊 **Diagrams** — Mermaid flowcharts render in-chat
- 💾 **History** — Save and resume conversations
- 📦 **Artifacts** — Auto-saved documents and outputs
- 🔬 **Research** — Auto-saved research sessions with sources
- ⚙️ **Settings** — Temperature, system prompt, API keys

### Commands
Type `/help` in chat to see all available commands.
